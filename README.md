# Integer to Roman Numeral Calculator

This application take integers as input and convert them into corresponding roman numerals


## How to Use

```bash
Clone Repository
```


In your terminal, run the following command:



```bash
yarn install
```

Or


```bash
npm install
```

## Run App

#### For development 
```bash
npm dev
```

#### For Production
```bash
npm build
```
and

```bash
npm start
```
